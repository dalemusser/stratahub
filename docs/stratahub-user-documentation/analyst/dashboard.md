# Dashboard

The **Dashboard** is an analyst's landing screen — an **Analyst Panel** giving a
read-only overview of the whole workspace.

## Summary cards

The cards show the totals across all organizations — **Organizations**, **Leaders**,
**Groups**, **Members**, and **Resources**. The figures are for reference only; an
analyst has no create or edit actions and no quick actions or admin tools.
