# Analyst guide

Comprehensive, feature-by-feature documentation for the **analyst** role in Strata
Hub. An analyst has **read-only** access for reporting — they can see an overview of
the whole workspace and export member data, but they don't create or change anything.

For a quick tour of the role, see [The Analyst View](../analyst-view.md); the pages
below are the full reference, in menu order.

## What an analyst can do

Analysts have **workspace-wide, read-only** access to reporting:

- See the **Dashboard** overview of every organization.
- Export member data from the **Members Report** (any organization).

They have no create, edit, or management tools — those belong to administrators and
coordinators.

## Features

- **[Dashboard](dashboard.md)** — a read-only overview of the workspace.
- **[Members Report](members-report.md)** — export member data as a CSV file.
- **[Announcements](announcements.md)** — your personal view of announcements.
- **[Profile](profile.md)** — your own account details, password, and theme.
- **[Public pages](public-pages.md)** — read the About, Contact, Terms, and Privacy
  pages.
