# Administrator guide

Comprehensive, feature-by-feature documentation for the **administrator** role in
Strata Hub. An administrator has full access to the workspace — managing
organizations, groups, leaders, members, content, and site-wide settings.

New here? Start with the workspace-setup walkthrough in
[Getting Started](../getting-started.md), then use these pages as a reference for
each feature.

## Navigation

Every admin screen shares the left navigation menu. Each item below opens one
feature; this guide documents them in menu order.

## Features

- **[Dashboard](dashboard.md)** — the admin landing page: workspace totals, quick
  actions, and admin tools.
- **[Activity](activity.md)** — live monitoring of who's online, plus the weekly
  summary and data export.
- **[Members Report](members-report.md)** — scope and export member data as a CSV
  file.
- **[Organizations](organizations.md)** — create, view, edit, and delete the schools
  or institutions in the workspace.
- **[Groups](groups.md)** — create groups and manage their users, resources, and
  apps.
- **[Leaders](leaders.md)** — create and manage leader (teacher) accounts.
- **[Members](members.md)** — create and manage member (student) accounts.
- **[Upload CSV](upload-csv.md)** — import members in bulk from a spreadsheet file.
- **[System Users](system-users.md)** — manage staff accounts (admins, analysts,
  coordinators).
- **[Resources](resources.md)** — create content for members and assign it to groups.
- **[Materials](materials.md)** — create content for leaders and assign it to them.
- **[Audit Log](audit-log.md)** — review a record of actions taken in the workspace.
- **[Sessions](sessions.md)** — see who's signed in and end sessions.
- **[Announce](announce.md)** — create announcements shown to users.
- **[MHS Dashboard](mhs-dashboard.md)** — follow members' progress through Mission
  HydroSci.
- **[MHS Builds](mhs-builds.md)** — view the catalog of Mission HydroSci build
  collections.
- **[Status](status.md)** — server health and diagnostics (description only).
- **[Public pages](public-pages.md)** — edit the About, Contact, Terms, and Privacy
  pages.
- **[Announcements](announcements.md)** — your personal view of announcements.
- **[Profile](profile.md)** — your own account details, password, and theme.
- **[Settings](settings.md)** — workspace name, logo, authentication methods, and
  footer.

## Not documented

- **Mission HydroSci** — the in-app player. No Mission HydroSci version is active in
  this workspace, so there's nothing to show; it's skipped for now.
