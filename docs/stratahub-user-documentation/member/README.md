# Member guide

Comprehensive, feature-by-feature documentation for the **member** role in Strata
Hub. A member is a student or participant with the simplest experience: sign in,
find the resources assigned to your group, and open them.

For a quick tour of the role, see [The Member View](../member-view.md); the pages
below are the full reference, in menu order.

## Features

- **[Resources](resources.md)** — your assigned resources and how to open them (your
  landing screen).
- **[Announcements](announcements.md)** — your personal view of announcements.
- **[Profile](profile.md)** — your own account details, password, and theme.
- **[Public pages](public-pages.md)** — read the About, Contact, Terms, and Privacy
  pages.
