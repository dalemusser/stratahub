# Developer documentation

Reference material for **developers** working with Strata Hub and the systems that
run alongside it (such as Mission HydroSci, StrataLog, and StrataSave).

**Developer is not a Strata Hub role.** When you're in Strata Hub you're signed in
under one of the normal roles — administrator, coordinator, leader, or member —
depending on what you're doing. These pages cover the things a developer needs to
understand that other users generally don't.

## Documents

- **[Object IDs](object-ids.md)** — how users, groups, organizations, and workspaces
  are identified by hex Object IDs, the reserved MHS developer sentinel user, and how
  to use these IDs in StrataLog and StrataSave.
